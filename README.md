# Personal Profile Card

A modern and responsive personal profile card built using HTML and CSS.

## Features

* Responsive profile card design
* Social media links (Facebook, X, Instagram, YouTube, GitHub, LinkedIn)
* Hover animations and transitions
* Premium UI with gradients and shadows
* Clean and modern layout

## Technologies Used

* HTML5
* CSS3

## About the Project

This project was created as part of my web development learning journey. It showcases my ability to build responsive layouts, work with CSS styling, and create visually appealing user interfaces.
